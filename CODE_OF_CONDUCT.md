# Code of Conduct / 行为准则

## Our Commitment / 我们的承诺

We are committed to making participation in this project a harassment-free
experience for everyone.  
我们致力于为所有参与者提供免于骚扰的协作环境。

## Expected Behavior / 期望行为

- Use welcoming and inclusive language.  
  使用友好、包容的表达方式。
- Be respectful of differing viewpoints and experiences.  
  尊重不同观点与经历。
- Give and accept constructive feedback gracefully.  
  以建设性方式提出并接受反馈。
- Focus on what is best for the community and project.  
  以社区和项目的共同利益为优先。

## Unacceptable Behavior / 不可接受行为

- Harassment, discrimination, or hateful language.  
  骚扰、歧视或仇恨性言论。
- Personal attacks, insults, or trolling.  
  人身攻击、侮辱或恶意挑衅。
- Publishing others' private information without consent.  
  未经同意公开他人隐私信息。

## Scope / 适用范围

This Code of Conduct applies to project spaces, including issues, pull requests,
and discussion channels tied to this repository.  
本行为准则适用于与本仓库相关的所有项目空间，包括 Issue、PR 和讨论区。

## Reporting / 反馈与举报

If you experience or witness unacceptable behavior, report it via the security
contact process in `SECURITY.md`.  
如你遭遇或目睹不可接受行为，请通过 `SECURITY.md` 中的安全联系流程进行反馈。

Project maintainers may take any action they deem appropriate in response to
behavior that violates this Code of Conduct.  
项目维护者可对违反本行为准则的行为采取其认为适当的措施。
